Getting Started

Make sure you have Nodejs and NPM installed on your computer.

NodeJS download link

Run in the Terminal

Use terminal and navigate to project root.

Then run npm install

Then run npm start

Build

Run npm run build to build the project. The build artifacts will be stored in the build/ directory.

Got to App.js component, then you found this code. If you chose home page version two. Then follow the setp. src/App.jsx

            <Switch location={location}>
                <Route
                    path="/contact"
                    component={ContactUS}
                />
                <Route
                    path="/home-three"
                    component={HomeThree}
                />
                <Route
                    path="/home-two"
                    component={HomeTwo}
                />
                <Route
                    path="/"
                    exact
                    component={HomeOne}
                />
